# Каталог data

Всё содержимое сервера хранится здесь:

- `my_world/` — текущий мир.
- `mods/` — `.jar` моды, которые грузит Forge.
- `config/` — конфиги модов и дополнительные настройки.
- `logs/` — логи сервера.

Перед запуском скопируй или создай файлы мира и модов. Этот каталог монтируется внутрь контейнера как `/data`.
