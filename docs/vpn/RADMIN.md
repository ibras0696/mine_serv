# Подключение через Radmin VPN

Если нет белого IP, можно собрать друзей в одной виртуальной сети.

## Создание сети
1. Скачай Radmin VPN с официального сайта и установи.
2. Нажми «Создать сеть», укажи имя (например, `forge-club`) и надёжный пароль.
3. Отправь эти данные друзьям.

## Подключение клиентов
1. Запусти Radmin VPN и выбери «Присоединиться к существующей сети».
2. Введи имя сети и пароль.
3. Дождись, пока рядом с ником появится статус «Зелёный» (соединение установлено).

## Настройка сервера Minecraft
1. Узнай IP, который Radmin VPN назначил хосту (обычно начинается с `26.`).
2. Пропиши его в `.env` в переменной `RADMIN_VPN_IP` — это будет подсказкой в README.
3. Игроки подключаются в Minecraft по адресу `26.x.x.x:25565`.

## Советы
- Держи Radmin VPN запущенным до тех пор, пока работает сервер.
- Если игрок не подключается, проверь, не блокирует ли его антивирус Radmin.
- Для постоянной работы лучше перейти на ZeroTier или реальный проброс порта у провайдера, но Radmin VPN подойдёт для быстрых игр с друзьями.
